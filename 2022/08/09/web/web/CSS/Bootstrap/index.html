流体容器 .container-fluid

流体容器，width：auto;加padding会向里缩，不会撑开盒子（盒子模型的公式）

备注：width:100%;加padding会撑开盒子

固定容器 .container

固定容器的底层是媒体查询

阈值 容器的width
大于等于1200（lg：大屏pc） 1170（1110+槽宽）
大于等于992（md:中屏pc），小于1200 970（940+槽宽）
大于等于768（sm：平板），小于992 750（720+槽宽）
小于768（xs：移动手机） auto

小于768时，相当于又是一个流体容器
